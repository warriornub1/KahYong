217. Contains Duplicate

Given an integer array nums, return true if any value appears least twice in the array, and return false if every element is distinct.

Example 1:

Input: num = [1,2,3,1]

Output: True

Example 2:

Input: num = [1,2,3,1]

Output: True

Python Solution

Solution

In a set(), there can be no duplicated items & the it takes O(1) to add into a set().

Loop through each value in the list using a for loop and append each element using a set() in Python.

If the element is already in the set(), that means that duplication exists. Return True

If the loop completes, that means that no duplicated element exists. Return False.

Time Complexity : O(n) as in the worst case scenario, the algorithm loops through each element in the list and O(1) for HashSet.add()

Memory Complexity : O(n) as in the worst case scenario, the algorithm adds all elements in the list into the set. 

          
def containsDuplicate(self, nums):
  hashset = set()

  for n in nums:
      if n in hashset:
          return True
      hashset.add(n)
  return False
